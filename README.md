🏠 House Price Prediction using Machine Learning

This project focuses on predicting house prices using machine learning techniques applied on a Kaggle dataset. The workflow includes data preprocessing, exploratory data analysis, model training, evaluation, and deployment using Streamlit.

📌 Project Overview

The goal of this project is to build a regression-based machine learning model that can predict house prices based on various features such as area, location, number of rooms, and other property attributes.

The project demonstrates an end-to-end data science pipeline from raw data to deployment.

Dataset Source:
Kaggle

🧠 Workflow
Data Collection (Kaggle dataset)
Data Cleaning and Preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Model Training
Model Evaluation
Deployment using Streamlit
📊 Exploratory Data Analysis
Analyzed feature distributions
Identified correlations between features and target variable
Handled missing values and outliers
Visualized key relationships using plots
🤖 Machine Learning Models Used

The following regression models were trained and evaluated:

Linear Regression
K-Nearest Neighbors (KNN) Regressor
Decision Tree Regressor
Random Forest Regressor
📏 Evaluation Metric
Root Mean Squared Error (RMSE) was used to compare model performance
The best-performing model was selected based on lowest RMSE score
🚀 Deployment

The final selected model was deployed as an interactive web application using Streamlit, allowing users to input house features and get predicted prices in real time.
